# 🛒 A-6 DigiTools

🔗 **Live Website:** https://digitools-webapp-toukir048.netlify.app/

A modern and responsive **Digital Tools platform** built using **React.js**. Users can browse digital products, view features, add items to the cart, and manage their purchases easily with a clean and user-friendly interface.

This project focuses on practicing **React fundamentals**, **state management**, **component structure**, **JSON data handling**, and **UI design** using Tailwind CSS & DaisyUI.

---

## 🚀 Live Features

- 🧭 Beautiful Navbar with cart count indicator  
- 🎯 Attractive Banner section with meaningful content  
- 📊 Stats section to display important information  
- 🔄 Toggle system between **Products** and **Cart**  
- 🃏 Product Cards in a 3-column responsive layout  
- 🛒 Add to Cart and Remove from Cart functionality  
- 🔔 Toast notifications using React-Toastify  
- 💰 Pricing section with structured plans  
- 🪜 Steps section explaining workflow  
- 🦶 Informative Footer section  
- 📱 Fully responsive design (Mobile, Tablet, Desktop)

---

## 🛠️ Technologies Used

- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🌼 DaisyUI  
- 🧠 JavaScript (ES6+)  
- 🔔 React-Toastify  
- 📄 JSON (for product data)

---

## 🛒 Cart Functionality

- Users can **add products** to the cart  
- Cart icon shows **total selected product count**  
- Users can **remove individual products**  
- **"Proceed to Checkout"** button clears the cart  

### 🔔 Toast Notifications

- Product added ✅  
- Product removed ❌  
- Checkout completed 💳  

---

## 🔄 Toggle Feature

### 📦 Products Button
- Displays all available products  
- Default active section  

### 🛒 Cart Button
- Displays selected cart items  
- Shows empty message if no items selected
